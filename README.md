# 🎯 Click Rush

# Live Link:
https://click-rush-fawn.vercel.app/

A fast-paced reaction-based clicking game built using **HTML, CSS, and JavaScript**.

The goal is simple: click the moving box as many times as possible before the timer runs out. Test your speed and reaction time!

## 🎮 Features

- ⏱️ 10-second gameplay timer
- 🎲 Random box movement
- 🌈 Random box colors
- 🏆 Score tracking system
- 🚫 Prevents multiple clicks on the same target
- 🔄 Automatic game reset after game over
- 🎮 Start button control
- 🎭 Game over screen

## 🕹️ How to Play

1. Click the **Start** button
2. A random box will appear inside the game area
3. Click the box to increase your score
4. The box changes position after every click
5. Try to get the highest score before the timer ends
